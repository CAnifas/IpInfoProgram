🌐 IP Info Tool

IP Info Tool — это консольная утилита на Python для быстрого получения подробной информации о любом IP-адресе в реальном времени. Программа позволяет узнать всё о местоположении, провайдере и сетевых характеристиках как своего собственного IP, так и любого другого адреса в интернете.
📋 Описание

Программа обращается к публичному API сервиса ipapi.co и отображает полную геолокационную и сетевую информацию по введённому IP-адресу. Результаты выводятся в аккуратно оформленной таблице прямо в терминале.

Утилита будет полезна системным администраторам, разработчикам, специалистам по кибербезопасности и всем, кому нужно быстро проверить информацию об IP-адресе без лишних действий — просто запустил и узнал.
⚙️ Возможности

    🔍 Поиск по любому IP — введи любой IPv4 адрес и получи полную информацию
    🏠 Определение своего IP — оставь поле пустым, и программа автоматически определит твой текущий IP-адрес
    🌍 Геолокация — страна, регион, город и почтовый индекс
    📍 Координаты — точные координаты (широта и долгота) для отображения на карте
    🕐 Часовой пояс — временная зона, привязанная к IP
    🏢 Провайдер и ASN — название интернет-провайдера и номер автономной системы
    ⚠️ Обработка ошибок — программа корректно сообщает об ошибках сети или неверном IP
    🔄 Бесконечный режим — можно проверять несколько адресов подряд без перезапуска

📊 Выводимая информация
Поле	Описание
IP	Проверяемый IP-адрес
Страна	Название страны и код (например, Russia, RU)
Регион	Регион / область
Город	Город
Почтовый	Почтовый индекс
Координаты	Широта и долгота
Часовой пояс	Временная зона (например, Europe/Moscow)
Провайдер	Название интернет-провайдера
ASN	Номер автономной системы
🚀 Запуск

python main.py

После запуска введи IP-адрес или нажми Enter для определения своего IP. Для выхода напиши exit.
🛠️ Требования

    Python 3.6+
    Подключение к интернету
    Сторонние библиотеки не требуются — только стандартная библиотека Python

📦 Упаковка в .exe (Windows)

Чтобы распространять программу как отдельный .exe файл с иконкой:

pip install pyinstaller
pyinstaller --onefile --icon=ipinfo_icon.ico IpInfo.py

Готовый файл появится в папке dist/.

Разработано для быстрой и удобной проверки IP-адресов прямо из терминала.
